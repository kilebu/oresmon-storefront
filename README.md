# Oresmon Storefront

A fully responsive gaming storefront homepage, built with HTML, Tailwind CSS,
and vanilla JavaScript — no framework, no build step.

**Live demo:** https://frontend-assignment-crux.vercel.app/

## What it demonstrates

- **Responsive layout** that holds together from mobile to widescreen
- **Visual consistency** — spacing, typography, and color applied as a system,
  not ad hoc
- **Design judgment over pixel-copying** — inspired by the Oresmon Tebex
  storefront, rebuilt with my own layout decisions
- **Lean vanilla JS** for interactions where a framework would be overkill

## Stack

HTML5 · Tailwind CSS · Vanilla JavaScript · Deployed on Vercel

## Run locally

Clone and open `index.html` in a browser. No dependencies, no build.
